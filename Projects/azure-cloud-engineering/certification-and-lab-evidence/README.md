## Evidence Approach

Evidence in this section may include:

- Lab README files
- Implementation summaries
- Architecture diagrams

- Command examples

---

## Confidentiality Note

This section should only contain lab, certification, or non-confidential learning evidence.

No workplace-sensitive information should be included, such as:

- Tenant IDs
- Subscription IDs
- Real user data
- Company information
- Internal URLs
- Client details
- Production screenshots
- Secrets, keys, or credentials

Any screenshots or examples should use lab environments, dummy data, or safely redacted information.
