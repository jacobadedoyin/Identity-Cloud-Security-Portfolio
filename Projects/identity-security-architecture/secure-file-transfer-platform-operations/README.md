# Secure File Transfer Platform Operations

## Overview

This project demonstrates secure file transfer platform ownership and administration using MOVEit Secure Transfer.

The work involved supporting secure transfer operations, managing user access, migrating existing file transfer solutions into MOVEit, designing a secure file hierarchy, supporting joiner / mover / leaver activity, and training users and stakeholders on how to use the platform effectively.

## Platform Covered

- MOVEit Secure Transfer

## Project Context

MOVEit was used as a secure file transfer platform for controlled movement of files between users, teams, and stakeholders.

This work sat within identity, access control, secure data movement, and secure environment operations because it involved controlling access to sensitive transfer areas and supporting safe file exchange processes.

## My Role

I owned MOVEit secure transfer administration responsibilities.

This included:

- Managing MOVEit user access
- Supporting secure transfer access requests
- Migrating existing transfer solutions into MOVEit
- Setting up a secure file and folder hierarchy
- Supporting joiner, mover, and leaver access activity
- Reviewing access requirements
- Supporting permission visibility
- Training users and more senior stakeholders on platform use
- Helping users resolve secure transfer access issues
- Supporting controlled movement of sensitive files
- Maintaining awareness of auditability, traceability, and secure data handling
