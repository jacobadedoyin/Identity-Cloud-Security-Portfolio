## Evidence Approach

This portfolio uses public-safe evidence to demonstrate practical capability without exposing confidential organisational information.

Evidence may include:

- Recreated diagrams
- Sanitised templates
- Public-safe certificates
- Lab evidence
- Concept architecture
- Workflow diagrams
- Work-based summaries with confidential details removed
- Reusable process documentation

No confidential organisational information is included.

This repository does **not** include:

- Real user data
- Client information
- Internal URLs
- Tenant IDs
- Production secrets
- Ticket references
- Real sign-in logs
- Internal SOPs
- Confidential operational procedures
