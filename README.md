# Cloud Security and IAM Portfolio

A professional portfolio demonstrating practical capability in cloud administration, identity and access management, Azure security foundations, secure data access, analytics platform support, and secure file transfer operations.

This repository combines certification-aligned learning, Azure lab work, and workplace-aligned examples. All evidence is public-safe, sanitised, and designed to demonstrate technical understanding without exposing confidential organisational information.

This portfolio demonstrates experience and practical understanding across:

- Secure data access architecture
- Qlik and Tableau platform access management

- Licence tracking, access reviews, and operational support
- Public-safe documentation, diagrams, templates, and lab evidence
