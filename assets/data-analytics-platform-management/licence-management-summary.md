# Licence Management Summary

## Overview

This evidence summary documents workplace-aligned licence management activity across Qlik and Tableau reporting platforms.

The work focused on maintaining visibility of assigned licences, supporting licence recovery, and helping ensure that platform access remained appropriate, controlled, and cost-aware.

All information in this document is sanitised, recreated, or summarised. No confidential organisational data is included.

---

## Purpose

The purpose of this evidence is to demonstrate practical experience in licence administration and access governance, including:

- Tracking assigned platform licences
- Supporting licence recovery from inactive or unnecessary accounts
- Maintaining visibility of licence usage
- Supporting joiner, mover, and leaver processes
- Reducing unnecessary licence allocation
- Supporting operational and financial accountability
- Linking licence management to access governance

---

## Platforms Covered

| Platform | Licence Management Activity |
| :--- | :--- |
| Qlik | Licence assignment visibility, user access review support, and licence recovery considerations |
| Tableau | Licence tracking, user access validation, and support for licence clean-up activities |

---

## Licence Management Activities

| Activity | Description |
| :--- | :--- |
| Licence tracking | Maintained visibility of which users had assigned platform licences |
| Licence recovery | Identified licences that could potentially be reclaimed when access was no longer required |
| Joiner support | Helped ensure new users received the correct level of platform access and licence assignment |
| Mover support | Supported licence and access changes when users moved roles or teams |
| Leaver support | Supported removal of access and recovery of assigned licences where appropriate |
| Access validation | Helped confirm whether licence allocation still matched business need |
| Support documentation | Created or maintained repeatable guidance for licence and access handling |

---

## IAM and Security Relevance

Licence management is closely linked to access governance because a licence often represents active access to a platform.

Maintaining licence visibility supports:

- Least privilege access
- Reduced stale access risk
- Improved user lifecycle control
- Better oversight of who can access reporting platforms
- Stronger operational governance
- More accountable platform administration
- Cost-aware access management
